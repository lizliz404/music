# Quiet Costs Money

**Genre:** Indie folk-punk / spoken-word adjacent  
**BPM:** ~112  
**Key:** Em  
**Mood:** Raw, ethnographic, cold-eyed then breaking, escape-bound

## Notes (songwriter)

- **Hook:** "Quiet costs money" — thesis line from the autoethnography ("安静是需要付费的"); entropy as default, silence as purchased good.
- **Arc:** Mother portrait → Father portrait → Self scarcity symptom → Cesspool refusal → Escape.
- **Images preserved:** dusty bilingual bookshelf (*Jane Eyre* / *Red and Black*), Pinduoduo hunger-eyes, triviality machine gun, ¥2.50 sweet-potato heart-palpitation, cesspool/toilet-door metaphor, escape velocity, open cage door.
- **Tone:** Cold ethnographic distance cracking into resolve — not pity-ballad, not revenge anthem.
