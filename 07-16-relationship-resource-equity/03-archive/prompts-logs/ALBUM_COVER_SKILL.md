# Album Cover Prompt Generator — skill excerpt for Cursor

Generate image prompts for album/song covers from finalized lyrics + style. NOT about generating images — about writing the prompt.

Output: 2 prompts per song
### A — Cover (1:1 square)
Bold, iconic, single-image statement. Works at thumbnail. Strong silhouette, clear focal point.

### B — Atmospheric (16:9 or 4:5)
Texture, lighting, implied motion/story. Banner/world.

Each prompt: 3–6 sentences dense visual language. Include subject/metaphor, composition, color palette (2–4 + accent), lighting, style/medium, aspect ratio, exclusions (no text).

Rules:
- Concept, not illustration of lyrics
- Single metaphor
- No typography
- Avoid stock clichés (stairs, lens flares, gold piles, motivational lighting)
- Negative space; asymmetry
- No figures unless essential
- Ask: pull without knowing song? emotional temperature? thumbnail? one focal point? embarrassing as real cover?

Before writing extract: 3 strongest concrete images, emotional temperature one word, palette, compositional device, exclusions.
