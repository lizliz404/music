### Style Field
- 4-7 meaningful descriptors. Sweet spot: 5-8 tags.
- Order: primary genre → subgenre/mood → tempo/energy → key instruments → vocal tone → production texture → dynamic arc
- Genre determines 60-70% of output. Lead with it.
- Never use artist names — they're filtered out. Describe the sound instead.
- Description field silently drops text past ~350 chars — be COMPACT.
- Use production tags: `polished clean mix`, `vintage analog warmth`, `close-miked vocals`, etc.

### Anti-Boring Devices (CRITICAL — Liz rejects "correct but mediocre" harder than wrong)
- Recurring 3-4 note guitar motif across intro/break/outro
- Pushed kick vs laid-back snare — heartbeat racing against walking pace
- Semitone drone bend — unresolved subliminal dread
- Strings on Final Chorus only — carry swell, exit before Outro
- Distant female wordless hum as memory, not companion
- 80-84 BPM instead of 92 — less sway, more weight
- Surface restraint + underlying tension (Eagles Hotel California × 李宗盛 山丘)

### Danger Tags to Avoid

- `folk-rock` — "rock" half triggers drum/bass aggression
- `warm` used 3+ times — pushes mix toward comfort, kills tension
- `soar` — triggers epic-rock mode
- `build the swell` — double volume-climb trigger
