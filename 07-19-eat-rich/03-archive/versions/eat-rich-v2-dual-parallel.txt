# 吃成富人 — v2 双版对照（Hermes 移植版 × Grok 极简版）

**Brief transparency**
- 两版各自独立跑了不同的 v2 路线，均基于同一份 22KB 专家评审（`01-source/eat-rich-dual-audit.txt`）。
- Hermes 版（subagent）：评审"最优路径"——B 骨架 + A 器官移植，以 A 叙事弧为主干。
- Grok 版（Cursor continue）：评审"路径 A"——极简狠版，砍到 ~2:30 冲短视频，保留自己的鉴宝装置做三处手术。
- 两版 brief 明确给了不同侧重，有意制造差异，不是收敛到同一答案。

## 硬伤修复对照

| 评审诊断 | Hermes 版 | Grok 版 |
|---|---|---|
| **硬伤 1：Chorus 重复结构** | 8 行真重复块 ×2 + Final 变奏；"鉴个宝"×6 一字不改 | 4 行真重复 ×3 一字不改；"嗨！"×6 |
| **硬伤 2：非语义钩子** | "——嗨！" ×12 + 保留"鉴个宝"祈使句（双钩并行） | "—— 嗨！" ×6（单钩，极简） |
| **隐喻兑换句** | "鉴出真货 就进腰包"（鉴定→值钱→富， chorus 内焊死） | "真货进账才算数——鉴完 我才吃成富人"（V2 尾行收束） |
| **"瘦羊" 修复** | "我以为自己在养生 其实在养膘"（更狠） | "包装比我还瘦"（更轻，保留原味） |
| **"被生意耍" 强化** | 升级为 Pre-Chorus 齐喊 tag ×2（melodic lift） | 保留 V2 尾行 + record scratch 强调 |
| **Bridge 假停顿** | [beat drop] + [silence] + [whispered] | [beat drop] + [silence] |
| **Outro** | A 版断绝式（退会员，行为改变完成） | 留白式（"喂 冰箱吗——" 被铃声打断） |

## 两版性格差异

| 维度 | Hermes 版 | Grok 版 |
|---|---|---|
| 体量 | 饱满（~3:10-3:30），Pre-Chorus + 8行 chorus | 极瘦（~2:20-2:40），无 Pre-Chorus，4行 chorus |
| 文学密度 | 高（A 版三金句全保留：加血/健身卡/弹簧） | 低-中（删了加血/健身卡/弹簧，保留拆迁/镜子/诺贝尔） |
| 洗脑工程 | 双钩（嗨！+ 鉴个宝）+ Pre-Chorus tag 三重轰炸 | 单钩（嗨！）极简重复 |
| 短视频切片点 | Chorus 任意 8 秒 + "被生意耍" tag | 全曲都是切片，Chorus 4 行一次成型 |
| 最佳单行 | 「我以为自己在养生 其实在养膘」/「是有人在你的胃口上装了弹簧」 | 「包装比我还瘦」/「鉴完 我才吃成富人」 |
| 风险 | Pre-Chorus + 8行 chorus 结构偏满，可能拖长 | 删掉了三个最强 punchline，文学性降档；"到底谁在坑我的胃口" 问句偏说理姿态保留 |
| 适用场景 | 完整收听 / 专辑曲 / KTV | 抖音 15 秒 / 短视频 BGM / 病毒传播 |

## 我的判断（Liz 参考）

两版是**同一首歌的两种发行格式**，不是竞争对手：

- **Grok 版是该先发的版本**——它是 15 秒切片时代的正确形态，把所有手术做到最狠，先用它试水温。
- **Hermes 版是完整收听版**——如果 Grok 版切片火了，Hermes 版就是"完整版音源"的答案（Pre-Chorus tag + 三个金句 + 弹簧 bridge + 断绝 outro）。
- 如果要**只做一版**：Hermes 版的结构更完整（评审两个硬伤都修到位 + "被生意耍"获得 Pre-Chorus 位置是神来之笔），Grok 版删金句删得有点狠（"给人生加血"和"胃口装弹簧"是评审点名的 A 版最强资产，Grok 版都丢了）。

**可混搭点**：Hermes 骨架 + Grok 的 "鉴完 我才吃成富人" 兑换句（比 "鉴出真货 就进腰包" 更直白有力）+ Grok 的 outro 留白（比退会员更有回味）。
