# 吃成富人 — v1 双版平行（Lyric × Grok 4.5）

**Brief transparency（两版独立性披露）**
- Lyric 版：由 Hermes 主 agent 撰写；加载了 songwriting-and-ai-music + liz-song-preferences（含 comedy-rap craft bank）+ 完整调研报告；**写之前已看到 Grok 版成品**（并行调度的物理限制——Cursor 先跑完）。为抵消"看过答案"的污染，Lyric 版刻意选了**不同的概念装置**（草根翻身 vs 专家鉴宝）、**不同的 chorus 句式**（威龙口号式 vs 鉴宝动作链）、**不同的叙事弧线**（俘虏→富人 vs 夜袭者→鉴宝师）。共通元素（便利店自嘲开场、"被生意耍"金句位、血糖过山车 bridge、电话/对话 outro）来自**同一份调研报告的结论**，属于同源推导，不是互抄。
- Grok 版：Cursor `--continue` 续接调研 session（grok-4.5-high）；brief 含 Liz 原话逐字 + 8 公式 + 5 建议 + craft 约束 + 交付格式；**未见过 Lyric 版**（真正的 blind）。
- 独立证据：两版概念装置、叙事视角、chorus 结构、verse 清单内容、bridge 收口方式全部不同（见文末对比表）。

## 对比笔记

| 维度 | A（Lyric） | B（Grok） |
|---|---|---|
| 概念装置 | 泡面俘虏→营养富人（草根翻身叙事） | 冰箱鉴宝师（专家人设，俯身鉴宝） |
| 叙事视角 | 第一人称失败者→翻身者，全程在战壕里 | 第一人称导师，翻身已完成，带你鉴宝 |
| Chorus 机制 | 上学威龙口号式（"除了真货我都没空吞"） | 野狼动作指令式（"打开冰箱 跟我鉴个宝"） |
| Chorus 收口 | "你也来？一起当富人"（邀请） | "鉴宝鉴宝 进我腰包"（自利） |
| V1 清单 | 信过的谣言+踩过的坑（代餐/健身卡/奶茶加血） | 夜袭日常+伪健康采购（外卖/果汁/泡面） |
| 干货金句 | 同一句"被生意耍"位 + Bridge「吃饱和吃对 是两笔账」 | 同一句"被生意耍"位 + Bridge 设问「谁在坑我的胃口？」 |
| Bridge | 陈述+拟人（胃口上装弹簧）+ 一句低声提醒 | 设问式（吃饱了吗吃对了吗）+ 甩出真凶问题 |
| Outro | 打电话给便利店退会员（断关系） | 打电话给冰箱报喜（和解） |
| 最佳单行 | 「健身房卡办了三年 就洗过两次澡」/「我在给人生加血」 | 「我还以为自己在瘦羊」/「镜子说：老铁你赢了，赢的是便利店」 |
| 时长感 | 偏满（~3:30+，V1 略长可砍 2 行） | 紧凑（~3:00-3:20，直接可生成） |
| 风险 | V1 黑历史略密，笑点密度高但信息密度跟着涨，注意别越界说教 | "鉴宝"梗依赖听众对鉴宝节目的熟悉度；bridge 末句问句略讲道理姿态 |
| 结尾风格 | 切断（退会员=行为改变完成） | 庆祝（已经富了=身份确认） |

**可混搭点**：B 的 chorus 更顺口、更适合跟唱；A 的 V1 清单更狠更笑；B 的 outro 更轻巧，A 的 outro 更闭环。若进 v2，值得试 B chorus + A V1 + B outro 的组合。
