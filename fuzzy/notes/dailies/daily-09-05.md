# 9.5

Tags: Daily

小区中的新猫子角色出现——也是雪豹色的，但颜色更浅，肉更多，更肥。看起来也是懒懒的那种，非常有意思。但我还是喜欢灰猫子，——“这猫太肥了”小女孩这个话真的是让我觉得好搞笑，但确实是肥。

对比下来还是喜欢灰猫子——瘦瘦的，毛也摸起来更舒服。阳光映衬着，真的看着还挺惬意。——他很粘人，可能已经认识我了。摸他脑袋时他总是一脸享受。非常有意思，虽然一摸手上全是毛，蹭来蹭去的很有意思。总之我挺享受的。

“哎呦，吓死我了你。”——享受那一刻，是一种惊喜的感觉。当时在看着肥猫子，想跟他交流一下。灰猫子就突然从我身后的时石台窜出来。

难得悠闲一刻——气温不是很热，阳光刚刚好，鸟明在耳边响着，自己在自说自话。没有太多的嘈杂，太多的垃圾声音。
