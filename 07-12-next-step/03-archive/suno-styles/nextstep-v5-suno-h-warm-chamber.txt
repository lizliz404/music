# "Next Step" v5 — Suno Ready Package
## Style H: Warm Chamber Folk-Pop
## BPM/Key: 92 BPM, D minor

### Style Prompt
```
Warm acoustic chamber folk-pop, soft piano and light string pads, gentle brushed percussion, warm double bass, restrained tenor male lead, tender unaffected delivery, genuinely melodic hook-driven chorus, soft dynamic swell through strings not vocal, intimate but polished production, 92 BPM D minor
```

### Exclude Styles
```
shoegaze, post-rock, ambient/electroacoustic texture, glitch, minimalism-for-its-own-sake, industrial, EDM, arena rock, orchestral bombast, belting, shouting, screaming, power ballad crescendo, gospel choir, cinematic triumph
```

### Sliders
Weirdness 15-25 · Style Influence 68-78
