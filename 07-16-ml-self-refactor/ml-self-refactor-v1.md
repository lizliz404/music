## Concept

A kid trained on the same few family labels every "bad model" child gets — lazy, careless, not like the good one — freezes those lines as identity. Adult life still runs the old script when the body goes soft. The song refuses both: destiny ("this is who I am") and victory-lap therapy ("I fixed myself"). Axis: **not defective — outdated version; residual fires on soft; still training.**

**Unified axis:** old weights / freeze vs update / residual on soft / version ≠ soul  
**Kill in sung lines:** model parameters, gradient, Bayesian, ACT jargon, Naval, "refactor complete" product copy  
**Keep as evidence:** dinner-table freeze, same three verdicts, flinch before thought, soft → old fire, small retrain step, not cured

## Craft notes (v1)

- **Hook protection:** `OLD WEIGHTS` gets a beat rest so it lands as event, not thesis glue.
- **Axis discipline:** ML stays as *weights / version / build / training* only where singable; family freeze + residual flinch carry the proof.
- **Bridge load:** one kitchen wound / one rewrite question / one leave-the-room act. No therapy checklist.
- **Ending:** residual honesty + still training. No "I chose growth."
- **Homogenization:** V1 "same three lines every problem kid gets" owns the shared childhood mold without sociology lecture.
