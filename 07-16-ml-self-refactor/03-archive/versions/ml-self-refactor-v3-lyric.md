## Concept

A kid who was labeled lazy/careless freezes that verdict as identity. Adult residual still fires in soft rooms. Not defective — outdated code still running after the labeler left. Still moving small weights; not a cure show.

**Axis:** old weights fire when soft · labeler gone / settings on · version ≠ soul  

**v3 design choices (audit signal used, not checklist):**
- Backbone: v2 images (labeler, court, need-too-much, weigh it now, map/country lightly)
- Chorus cadence: v1 short lines + beat rest after OLD WEIGHTS
- Bugs fixed: no Wound:/Question:/Act: as lyrics; no "table light"; money metaphor thinned
- Tech budget: keep *weights / fire / soft / settings / training* sparingly; body + kitchen carry the rest
- Ending: residual, song-specific — **not** the archived Impotent couplet "Not cured. Not pure."
- Cold open: 1 signature line (audits agreed pre-hook helps)
- Bridge act: small, slightly clumsy (stood / sat / stood) — not heroic exit
- Second person: light present "you" (kindness that triggers flinch) without turning into a love song
